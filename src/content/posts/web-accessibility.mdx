Web accessibility is not optional. Let's explore how to make your websites accessible to everyone.

## Semantic HTML

The foundation of accessible websites:

```html
<header>
  <nav>
    <ul>
      <li><a href="#home">Home</a></li>
    </ul>
  </nav>
</header>
```

## ARIA Labels

When HTML semantics aren't enough:

```html
<button aria-label="Close menu">
  <svg>...</svg>
</button>
```
