## Prerequisites

- Node.js and npm/pnpm
- Basic JavaScript knowledge
- A code editor

## Setting Up Your First Project

Using Vite is now the recommended way to start a new React project. Here's how:

```bash
pnpm create vite my-react-app --template react-ts
```
